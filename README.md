# Rails Page.js Demo


## Installation

	rake db:create db:migrate db:seed
	rails s


### Configuration
```ruby
#app/assets/javascript/application.js
//= require page
#config/initializers/assets.rb
Rails.application.config.assets.precompile += %w( page.js )
```
And then execute:

  $ bundle
